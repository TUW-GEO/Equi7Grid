# Basic mode

The basic mode restricts the set of tools to a minimum to focus on display and map interaction. When clicking on "Basic" the toolbar on the left side of the map will display the following icons:

- Home view (🏠): Pans and zooms the map view to the home view after and displays all seven Equi7Grid zones.
- Zone toggle (🧱): Toggle for displaying the 7 continental zones.
- Tiling toggle (🔷): Toggle for displaying a basic tiling set (can be changed in the expert mode with the [layer manager](./layers.md) (📚)).
- [Display settings](./settings.md) (🔩): Change transparency and stroke width of the tilings.

:::{note}
The [layer manager](./layers.md) (📚) defines what tilings are displayed, which is synchronised with 🔷 in the "Basic" mode. Thus, when switching from "Basic" to "Expert" without displaying tilings (🔷 was not clicked) and then back to "Basic" again, 🔷 does not do anything, because no tiling is set visible in the layer manager.
:::
